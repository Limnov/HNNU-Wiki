# 人工智能学院简介

## 学院概况

海南师范大学人工智能学院（原信息科学技术学院）是学校重点建设的工科学院之一，致力于培养人工智能、计算机科学、软件工程等领域的高素质应用型人才。

## 历史沿革

- **信息科学技术学院**：原学院名称，涵盖计算机相关专业
- **人工智能学院**：为适应人工智能发展需要，学院转型升级

## 专业设置

### 本科专业

**计算机科学与技术**（理学/师范类）
- 培养目标：中小学信息技术教育师资和计算机应用人才
- 主修课程：
  - 程序设计基础
  - Python 语言程序设计
  - 数据结构
  - 计算机组成原理
  - 操作系统
  - 计算机网络
  - 数据库原理及应用

**软件工程**（工学）
- 培养目标：软件开发工程师和软件项目管理人才
- 特色项目：与印度国家信息技术学院（NIIT）共建
- 主修课程：
  - 软件工程基础
  - 系统设计与分析
  - 软件测试
  - Web 开发技术
  - 移动应用开发
- 国际认证：NIIT 软件工程师认证

**网络空间安全**（工学）
- 培养目标：网络安全和信息安全专业人才
- 主修课程：
  - 网络安全基础
  - 密码学
  - 网络攻防
  - 信息安全审计

**人工智能**（新增专业）
- 培养目标：AI 算法工程师和智能应用开发人才
- 主修课程：
  - 机器学习
  - 深度学习
  - 自然语言处理
  - 计算机视觉
  - 神经网络与 TensorFlow 实践

## 学科实力

### 教学平台
- 数据科学与智慧教育教育部重点实验室
- 计算机实验教学示范中心
- 软件工程实训基地

### 师资力量
- 教授、副教授占比
- 博士学位教师占比
- NIIT 国际认证讲师

## 办学特色

### 1. 师范培养优势
- 承担海南省中小学信息技术师资培养任务
- 师范类专业学生可获取教师资格证
- 教育实习基地覆盖全省中小学

### 2. 国际合作办学
- 与印度 NIIT 合作软件工程专业
- 引入国际课程体系
- 双证书培养模式（学历证书 + 国际认证）

### 3. 产教融合
- 与企业共建实训基地
- 项目驱动式教学
- 校企双导师制

## 学生发展

### 就业方向
- **教育系统**：中小学信息技术教师、教育技术管理
- **互联网企业**：软件开发工程师、算法工程师、测试工程师
- **政府机构**：信息化管理、网络安全管理
- **金融机构**：银行、证券公司 IT 部门
- **继续深造**：考研、出国留学

### 就业率
- 本科生就业率：约 94%
- 研究生就业率：100%

### 升学情况
- 考研率：约 7%
- 保研率：约 1.7%（全校平均）
- 保研高校：985、211 高校及本校
