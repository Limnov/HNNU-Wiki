# Microsoft Office 学生优惠

## 优惠内容

### 免费（学校提供）

如果学校提供，可直接使用：
- **Microsoft 365 教育版** - 完全免费

### 个人购买优惠

- **Microsoft 365 Personal** - **50% off** 折扣
- **部分产品** - **10% off** 折扣
- **Surface 硬件** - **2 折** 优惠
- **Xbox** - **8 折** 优惠

## 申请方式

1. 使用教育邮箱（.edu）注册
2. 访问 [Microsoft Education](https://www.microsoft.com/zh-cn/education/products)
3. 验证学生身份
4. 享受折扣价格

## 实用技巧

### 学校未提供免费版本

- 如果学校未提供 Office 365，可购买教育版享受 **50% off**
- **Microsoft 365 Personal** 原价 ¥399/年，学生价约 **¥200/年**

### 免费替代方案

- **WPS Office** - 国产办公软件，有免费版
- **ONLYOFFICE** - 在线办公套件，学生 30% 折扣
- **Google Docs** - 完全免费，需科学上网
- **LibreOffice** - 完全免费开源

## 教育邮箱的其他用途

.edu 邮箱可用于申请其他学生优惠：
- GitHub 学生开发包
- Azure 云服务
- Notion 免费使用
- 各种开发工具的学生版
