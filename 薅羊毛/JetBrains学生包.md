# JetBrains 学生包

## 免费权益

通过学生认证后，可**免费使用**所有 JetBrains 专业开发工具：

- **IntelliJ IDEA** - Java 开发
- **PyCharm** - Python 开发
- **WebStorm** - Web 前端开发
- **PhpStorm** - PHP 开发
- **CLion** - C/C++ 开发
- **DataGrip** - 数据库管理
- **AppCode** - iOS/macOS 开发
- **GoLand** - Go 语言开发
- **RubyMine** - Ruby 开发

## 申请条件

- ✅ **在校学生**（正规教育机构）
- ✅ **教育邮箱**（.edu 邮箱）
- ✅ **学生证明文件**（学生证、录取通知书等）

## 申请步骤

1. 访问 [JetBrains Student Pack](https://www.jetbrains.com.cn/academy/student-pack/)
2. 点击"申请 Student Pack"
3. 使用教育邮箱注册账号
4. 上传学生证明文件
5. 等待审核（通常 1-3 个工作日）
6. 审核通过后即可下载使用

## 使用时长

- **有效期**：1 年
- **续期**：每年需重新认证

## 实用建议

- 适合计算机专业学生
- 用于课程学习、项目开发
- 准备竞赛和面试
- 搭建个人项目
