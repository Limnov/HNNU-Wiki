## 📖 关于手册

本手册第一版发布于 2024 年 10 月 22 日,至今仍在持续更新中。

我们深知一本手册无法帮助大家深入了解某一个领域的知识或掌握某项技能,但希望能为你的大学生涯提供一些方向和参考,帮助你找到属于自己的路。
