# 编程学习路线图

> 选择比努力更重要。找到适合自己的方向，然后坚持走下去！

## 🎯 如何选择方向

### 第一步：了解各个方向

```
前端开发 💻      →  做网站、小程序、H5
后端开发 🔧       →  处理数据、API、服务器
移动开发 📱       →  App开发（iOS/Android）
人工智能 🤖       →  机器学习、深度学习
算法竞赛 🏆       →  ACM/ICPC、OI
游戏开发 🎮       →  游戏引擎、游戏逻辑
数据分析 📊       →  数据处理、可视化
```

### 第二步：评估兴趣和能力

**兴趣自测：**
- 喜欢看到视觉效果 → 前端开发
- 喜欢逻辑思考 → 后端开发
- 对数学感兴趣 → AI/算法
- 喜欢做应用 → 移动开发
- 热爱游戏 → 游戏开发

**能力匹配：**
- 逻辑强、数学好 → AI/算法
- 审美好、有创意 → 前端
- 沟通能力强 → 全栈
- 追求极致性能 → 后端/C++

### 第三步：考虑就业前景

| 方向 | 起薪 | 就业难度 | 发展前景 |
|------|------|---------|---------|
| 前端 | ⭐⭐⭐ | ⭐⭐ | 稳定 |
| 后端 | ⭐⭐⭐⭐ | ⭐⭐⭐ | 优秀 |
| AI | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 爆发 |
| 移动端 | ⭐⭐⭐ | ⭐⭐⭐⭐ | 趋缓 |
| 算法 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 精英 |

## ❓ 常见问题

### Q1: 零基础应该从哪门语言开始？
**A**: 推荐从 **Python** 开始，语法简单，上手快，应用广。

### Q2: 每天应该学多久？
**A**:
- 兴趣阶段：1-2小时/天
- 入门阶段：2-3小时/天
- 进阶阶段：3-4小时/天
- 保持比时长更重要

### Q3: 看视频还是看书？
**A**:
- **初学者**：看视频更直观
- **有基础后**：看书更系统
- **最佳方案**：视频入门 + 书籍深入

### Q4: 遇到bug怎么办？
**A**:
1. 仔细阅读错误信息
2. Google/百度搜索
3. Stack Overflow 搜索
4. 问 AI 编程助手
5. 问技术社区

### Q5: 什么时候可以找工作？
**A**:
- **前端**：3-6个月系统学习 + 2-3个项目
- **后端**：6-12个月学习 + 实习经验
- **AI**：1-2年学习 + 硕士学历更好

### Q6: 需要考研吗？
**A**:
- **不需要**：前端、后端、产品
- **建议考**：AI、算法、国企
- **看公司**：大厂更看重能力
