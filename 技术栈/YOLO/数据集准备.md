# YOLO 数据集准备

## 数据集目录结构

```
datasets/
└── my_dataset/
    ├── images/
    │   ├── train/
    │   │   ├── img1.jpg
    │   │   └── img2.jpg
    │   └── val/
    │       └── img3.jpg
    └── labels/
        ├── train/
        │   ├── img1.txt
        │   └── img2.txt
        └── val/
            └── img3.txt
```

## 标注文件格式

每个 `.txt` 文件对应一张图片，每行一个目标：

```
class_id center_x center_y width height
0 0.5 0.5 0.3 0.4
1 0.2 0.3 0.1 0.2
```

> 坐标值必须归一化到 0-1 范围

## 配置文件 data.yaml

```yaml
path: ./datasets/my_dataset
train: images/train
val: images/val

nc: 2                    # 类别数量
names: ['cat', 'dog']    # 类别名称
```

## 标注工具推荐

| 工具 | 特点 |
|------|------|
| LabelImg | 简单易用，本地运行 |
| CVAT | 功能强大，团队协作 |
| Roboflow | 在线标注，自动导出 YOLO 格式 |

安装 LabelImg：

```bash
pip install labelImg
labelImg
```

## 使用 Roboflow（推荐）

Roboflow 提供一站式数据管理：

1. 上传图片
2. 在线标注
3. 数据增强
4. 导出 YOLO 格式

```python
# 从 Roboflow 下载数据集
from roboflow import Roboflow
rf = Roboflow(api_key="YOUR_API_KEY")
project = rf.workspace().project("my-project")
dataset = project.version(1).download("yolo11")
```

## 数据要求

- 每个类别至少 100+ 张图片
- 尽量覆盖各种场景和角度
- 训练集:验证集 ≈ 8:2 或 7:3
- 图片分辨率建议 640x640 以上

## 快速测试数据集

Ultralytics 提供示例数据集：

```python
from ultralytics import YOLO

model = YOLO('yolo11n.pt')
# 使用 coco8 示例数据集
model.train(data='coco8.yaml', epochs=10)
```
