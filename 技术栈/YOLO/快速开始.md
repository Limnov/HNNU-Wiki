# YOLO 快速开始

## 加载预训练模型

```python
from ultralytics import YOLO

# YOLO11（推荐入门）
model = YOLO('yolo11n.pt')

# YOLO26（最新，端到端无NMS）
model = YOLO('yolo26n.pt')
```

## 模型规模选择

| 模型 | 速度 | 精度 | 参数量 | 适用场景 |
|------|------|------|--------|----------|
| yolo11n / yolo26n | 最快 | 最低 | ~2.6M | 移动端、边缘设备 |
| yolo11s / yolo26s | 快 | 较低 | ~9.4M | 实时应用 |
| yolo11m / yolo26m | 中等 | 中等 | ~20M | 平衡选择 |
| yolo11l / yolo26l | 较慢 | 较高 | ~25M | 精度优先 |
| yolo11x / yolo26x | 最慢 | 最高 | ~57M | 追求精度 |

## 检测图片

```python
# 检测单张图片
results = model('image.jpg')

# 显示结果
results[0].show()

# 保存结果
results[0].save('output.jpg')
```

## 检测视频

```python
# 视频文件
results = model('video.mp4', save=True)

# 摄像头实时检测
results = model(source=0, show=True)
```

## 获取检测结果

```python
results = model('image.jpg')
result = results[0]

# 遍历检测结果
for box in result.boxes:
    cls = int(box.cls[0])      # 类别ID
    conf = float(box.conf[0])  # 置信度
    xyxy = box.xyxy[0]         # 边界框 [x1,y1,x2,y2]

    print(f"类别: {cls}, 置信度: {conf:.2f}")
```

## 调整检测阈值

```python
# 置信度阈值（默认0.25）
results = model('image.jpg', conf=0.5)

# IoU 阈值（默认0.7，YOLO26无需设置）
results = model('image.jpg', iou=0.5)
```

## 支持的任务

| 任务 | 模型文件 | 说明 |
|------|----------|------|
| 目标检测 | yolo11n.pt | 检测物体边界框 |
| 实例分割 | yolo11n-seg.pt | 分割物体轮廓 |
| 图像分类 | yolo11n-cls.pt | 图像分类 |
| 姿态估计 | yolo11n-pose.pt | 人体关键点 |
| 旋转框检测 | yolo11n-obb.pt | 航空影像等 |

```python
# 分割示例
model = YOLO('yolo11n-seg.pt')
results = model('image.jpg')

# 姿态估计示例
model = YOLO('yolo11n-pose.pt')
results = model('person.jpg')
```
