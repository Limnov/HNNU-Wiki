# YOLO 简介

## 什么是目标检测？

目标检测 = **定位** + **分类**

- 定位：找出物体在图像中的位置（边界框）
- 分类：识别物体是什么类别

## YOLO 是什么？

YOLO（You Only Look Once）是实时目标检测算法，核心思想是把检测任务当作回归问题，一次前向传播完成检测。

### 发展历程

| 版本 | 发布时间 | 特点 |
|------|----------|------|
| YOLOv1-v3 | 2015-2018 | 奠基版本，引入多尺度检测 |
| YOLOv4-v5 | 2020 | Mosaic增强、易用性提升 |
| YOLOv6-v7 | 2022 | 工业优化、姿态估计 |
| YOLOv8 | 2023 | Ultralytics 重构，全能视觉AI |
| YOLOv9-v10 | 2024 | PGI、GELAN、无NMS |
| **YOLO11** | 2024.09 | 更少参数，更高精度 |
| **YOLO26** | 2026.01 | 端到端无NMS，边缘优化 |

### YOLO11 新特性

- 参数减少 22%，精度更高
- 支持检测、分割、分类、姿态估计、旋转框
- 跨平台部署（边缘设备、云平台）

### YOLO26 新特性

- **端到端无NMS**：直接输出结果，无需后处理
- **CPU速度提升 43%**：专为边缘设备优化
- **MuSGD优化器**：更稳定的训练
- **开放词汇检测**：YOLOE-26 支持任意类别

### 核心优势

- **速度快**：可达 30+ FPS，支持实时检测
- **端到端**：无需复杂的后处理（YOLO26）
- **易部署**：支持多种硬件和格式

## 适用场景

- 安防监控
- 自动驾驶
- 工业检测
- 医疗影像
- 智能零售
- 机器人视觉

## 安装

```bash
pip install ultralytics
```

验证：

```python
from ultralytics import YOLO

# YOLO11
model = YOLO('yolo11n.pt')

# YOLO26（最新）
model = YOLO('yolo26n.pt')

print("安装成功！")
```
